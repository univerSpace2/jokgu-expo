1. 플레이어 및 팀 구성 관련 기능

- 플레이어 관리
  - 플레이어 추가 기능: 앱 사용자(플레이어)를 추가할 수 있음.
  - 플레이어 정보 수정 및 삭제 기능: 이미 등록된 플레이어 정보에 대해 수정이나 삭제를 할 수 있음.
- 팀 구성
  - 최대 4팀까지 구성 가능: 플레이어들을 이용하여 팀을 구성.
  - 각 팀 구성원 수정: 필요에 따라 팀에 포함된 플레이어를 변경하거나 수정할 수 있음.

⸻

2. 게임 진행 화면

- 게임 시작 및 설정
  - 최소 2팀 이상 선택하여 게임 진행.
  - 최대 3세트까지 게임 세팅 가능.
  - 세트당 점수 한도: 각 세트에서 최대 21점까지 설정.
- 실시간 경기 진행
  - 점수 기록 및 표시: 경기 중 점수를 실시간으로 기록하고 화면에 표시.
  - 무르기(리와인드) 기능: 잘못된 입력 시 점수를 취소할 수 있음.
  - 득점 로그 기록: 득점 시 로그를 쌓아 추후 분석이나 기록으로 활용.
- 내기 및 정산
  - 내기값(벌칙, 회비 등) 설정: 게임 전 내기(벌칙) 관련 금액이나 규칙 설정.
  - 게임 종료 시 정산 기능: 경기 종료 후 팀별로 벌칙을 적용하고, 정산 내용을 처리.

⸻

3. 모임 및 플레이어 정산 화면

- 그룹/모임 구성
  - 그룹 생성: 사용자들이 모임이나 그룹 단위로 관리할 수 있도록 그룹 생성.
  - 그룹원 추가: 그룹에 참여할 구성원들을 추가.
  - 일일 모임 생성: 특정 날짜 및 시간에 모임을 구성하고, 해당 모임에 참여할 그룹원들을 선택.
- 새 게임 진행 시
  - 참여한 그룹원들 중에서 팀 구성 진행: 참여 인원으로 팀을 구성하고 게임 설정(예: n판 m선승, 승리 점수, 벌칙 등)을 마친 후 점수 카운트 화면으로 이동.
- 모임 종료 후
  - 구성원별 벌칙 정리: 게임 종료 후 패배 팀 구성원에게 벌칙 부여.
  - 경기 및 벌칙 내용 저장: 해당 모임의 경기 내역과 벌칙 내용을 기록.
  - 결과 공유 기능: 이미지나 클립보드로 정산된 내용을 공유할 수 있음.

4. 전체 메뉴 구성

- 모임 시작하기
  - 기존 모임 불러오기
    - 목록 뷰를 통해 기존 모임 확인: 진행된 게임 정보, 장소, 시작~종료 시간, 날짜 등 표시.
- 새 모임 시작하기
  - 기본 정보 입력: 날짜, 시작 시간, 장소.
  - 구성원 선택: 모임에 참여할 구성원 선택.
  - 게임 진행 목록
    - 새 게임 시작하기
      - 팀 갯수 및 팀 당 구성원 설정.
      - 게임 횟수 설정: n판 m선승 방식.
      - 승리 점수 설정.
      - 벌칙 설정: 내기, 회비, 맞기 등 여러 옵션.
      - 게임 플레이
        - 점수 카운트 화면 (무르기 기능 포함).
        - 득점 로그 기록(재미 요소로 활용 가능).
        - 세트 변경 및 목표 점수 도달 시 경기 종료.
      - 게임 종료 후: 패배 팀 구성원에 대한 벌칙 부여, 경기 내용 및 벌칙 내용 저장.
    - 게임 정보 보기 - 진행된 게임의 팀 구성, 벌칙 내용, 경기 기록 등을 확인할 수 있음.
- 구성원 관리하기
  - 구성원 추가, 삭제, 정보 수정 기능 제공.
- 족구장 모아보기
  - 전국 각지의 족구장 정보 제공: 장소, 예약 필요 여부, 예약 방법, 가격 정보 등 포함.
